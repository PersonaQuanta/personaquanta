Executive overview

PersonaQuanta is an explainable self-model experience for users: it converts inputs into drivers, trade-offs, and trajectory — built to give users a reason to finish the flow.

Pseudonym-first

  • No real name required
  • Inputs are user-controlled
  • Designed to reduce friction at step 1

Explainable “Top Drivers”

  • Plain-language drivers (why the result moved)
  • Constraints highlighted (what limits the upside)
  • Built to reduce “black box” distrust

CONF + ranges

  • Range-first outputs (not single-point certainty)
  • CONF tied to input completeness/consistency
  • Missing data widens ranges transparently

What-if exploration

  • One variable change → directional impact
  • Users learn trade-offs safely
  • Creates a “finish to see the result” loop

Pilot measurement (value capture)

  • Drop-off by step (where users quit)
  • Completion rate (finish questionnaire)
  • Time-to-first-value (first meaningful output)
  • What-if engagement (did users explore trade-offs)

Metrics are measured during the pilot; outcomes are not guaranteed.

90-second reference flow

  1. Input: short questionnaire (pseudonym-first)
  2. Drivers: top drivers + constraint in plain language
  3. One what-if: tweak one variable → direction + confidence change
  4. Summary: exportable report format (template)
